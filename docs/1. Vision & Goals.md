# Vision & Goals — Showrades

**Version:** 1.0
**Last Updated:** 2026-06-12
**Status:** 🟢 Living document — reflects current build direction (P0 game loop built)

> **"A single screen where a 5-year-old and a grandparent can play charades together — no reading required."**

---

## 1. Vision

Showrades is a free, open-source, mobile-first web app that brings generations together for game night with zero friction — no app store downloads, no accounts, no ads. Open one link, split into teams, start acting.

It solves the core headache of mixed-age game nights: players aged **7 and under** use **Picture Charades** (image/emoji prompts, no reading required), while players aged **8 and up** use **Word Charades** (text prompts). Both modes share the same game flow, scoring, and UI — nobody gets left out, and a 5-year-old and a grandparent can play on the same team, side by side.

## 3. User Personas

### 🧒 Young Player (Age 4–7)
Cannot read fluently; relies on pictures and icons. Needs large, colorful visuals and simple UI cues. May be guided by a parent to interact with the screen.

### 👦 Older Child / Tween (Age 8–12)
Comfortable reading short words. Enjoys a challenge; appreciates category variety. Likely the one holding the phone.

### 👨‍👩‍👧 Adult / Parent
Sets up the game and manages scoring. Wants minimal friction to get the game started. May play against or alongside kids — and acts as a "scaffolding" co-player for younger kids (see Section 5).

### 👴 Grandparent / Extended Family
Needs large text and clear navigation. Unfamiliar with complex UI patterns; relies on intuitive flow.

## 5. Direction: Sneaky-Educational Design

Beyond "just a game," Showrades aims to layer in developmental and educational value without making it feel like homework:

- **Dual-coding for pre-readers:** pairing a visual/emoji prompt with its word reinforces word-symbol association for kids transitioning from Pictures Mode to Words Mode.
- **Secret Clues for parents:** a short, descriptive hint a parent/caregiver can read aloud to a pre-reader — turns the adult into an active co-player ("clue-giver") rather than a passive phone-holder.
- **Educational Twists:** a one-line fact tied to each word (biology, physics, geography, etc.) — surfaced optionally (e.g. on the round summary screen) so kids pick up incidental learning between rounds.

This is a **direction, not a locked spec** — it needs further refinement on *where* in the UI these surface and *how often*. `Research on contents.md` contains the research basis and a fully worked example database (200 items across 8 categories with Secret Clues + Educational Twists) that this direction should build on **top of** the current 6 locked categories — see `2. Backlog & Status.md` (P2 — Expanded content packs) for how this is tracked.

## 6. Content Strategy

**Locked launch categories (6):** Animals (mint), Food (sun), Fruits (coral), Jobs (sky), Veggies (plum), Sports (sky) — `lib/data/words.ts`, 98 word cards, emoji-based visuals for Picture Charades.

**Future expansion direction (on top of the 6, not replacing them):** additional categories from `Research on contents.md` — Vehicles, Everyday Stuff, Nature, TV & Movies, Actions — each carrying the richer per-item metadata (illustration prompt, Secret Clue, Educational Twist) described in Section 5.

## 7. Market Positioning

Most digital charades/guessing games fall into two camps: **app-store downloads** (Heads Up!, Charades!™ — friction for toddlers and tech-shy grandparents) or **bare-bones web tools** (Playmime, Who Am I? — text/GIF-only, no illustration mode for pre-readers, no multi-generational scaling).

Showrades' white space:

1. **Dual-Age Engine** — one UI/scoring system where a 5-year-old gets full-screen illustrated prompts (with parent Secret Clues) while an older sibling or grandparent gets crisp large-text prompts, on the same team.
2. **Zero-friction web app** — runs instantly in a mobile browser, no downloads/sign-ups/ads.
3. **Sneaky-educational focus** — built around developmental benefits (motor skills, biomimicry, vocabulary), not pop-culture trivia.
4. **Offline resilience** — works after first load for family gatherings, camping, patchy Wi-Fi.
