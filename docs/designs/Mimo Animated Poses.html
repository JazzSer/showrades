01 — The Set

Five moods, mapped to moments

Each mood pairs a brand pose with a looping animation tuned to where it appears in the pass-and-play loop. Mimo's body colour carries the mood — and the brand palette — straight through the game.

Idle
Home screen — a calm breathing bounce & blink
mood="idle"
Thinking
Timer running — sways, looks up, "?" bubbles
mood="thinking"
Cheer
Correct answer — jumps with a sparkle burst
mood="cheer"
Dizzy
Timeout or wrong — woozy wobble, orbiting stars
mood="dizzy"
Wave
Pass-and-play handoff — a friendly wave
mood="wave"

03 — Drop-in

One file. One tag.

Mimo ships as a single self-contained web component. Add the script once, then place the element anywhere — markup, React, anything that renders HTML. Styles and IDs are sealed inside its shadow root, so nothing on the host page can collide.

<!-- 1 · include once -->
<script src="mimo/mimo-mascot.js"></script>

<!-- 2 · drop in wherever a moment calls for Mimo -->
<mimo-mascot mood="idle"></mimo-mascot>      <!-- home -->
<mimo-mascot mood="thinking"></mimo-mascot>  <!-- timer -->
<mimo-mascot mood="cheer" size="220"></mimo-mascot>
<mimo-mascot mood="dizzy"></mimo-mascot>
<mimo-mascot mood="wave"></mimo-mascot>

<!-- 3 · drive it from game state -->
const mimo = document.querySelector('mimo-mascot');
onCorrect(() => mimo.setAttribute('mood', 'cheer'));
onTimeout(() => mimo.setAttribute('mood', 'dizzy'));
Zero dependencies
Pure CSS + inline SVG inside a shadow root. No framework, no build step, no runtime — works in any HTML page.
Friendly aliases
correct, wrong, timeout, happy, pass all resolve to the right mood — name it however your code reads best.
Reduced-motion safe
Honors prefers-reduced-motion automatically, freezing each mood on a clear, expressive pose.

04 — Reference

Attributes

Everything is controlled with plain HTML attributes — set them statically or change them at runtime.

AttributeDescriptionValues
moodWhich pose + animation to show. Drives both expression and body colour.idle thinking cheer dizzy wave
sizeRendered height in pixels. Width scales automatically.any number — default 180
stillBoolean. Freezes the mascot on its end-state with no looping motion.present / absent

05 — Accessibility

Still, but never lifeless

When a player has reduced motion turned on — or when you set still yourself — every animation switches off and Mimo rests on the most readable frame of each mood. The mood still reads instantly; it simply holds its pose.

Animated
Jumps and sparkles on every correct answer.
Reduced motion / still
Holds the cheer pose — arms up, sparkles in place.
